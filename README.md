pywebtex
========

A pyramid web application for editing latex.

### What Works - so far

- File upload
- Create projects
- Compile files
- View compile log
- Save files


### What doesn't yet work

- Create subfolders
- User management
- Dropbox sync
- Git sync and version control
- Live multi-user edit (via websockets)
- Document sharing
